# 🌍 CodeAlpha Language Translator

A Flask-based Language Translation Tool that translates text between multiple languages using the Google Translator API.

## 📌 Features

- 🌐 Translate text between multiple languages
- 🎨 Simple and responsive user interface
- ⚡ Fast translation using Google Translator API
- 📋 Easy to use
- 🔄 Supports multiple language pairs

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- deep-translator

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslator.git
```

### Go to project folder

```bash
cd CodeAlpha_LanguageTranslator
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open in Browser

```
http://127.0.0.1:5000
```

## 📖 How It Works

1. Enter text.
2. Select source language.
3. Select target language.
4. Click **Translate**.
5. The translated text is displayed instantly.
